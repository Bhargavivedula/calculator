# 🧮 CALCU PRO — Advanced Smart Calculator

CALCU PRO is a modern and responsive smart calculator built using HTML, CSS, and JavaScript.

It combines:
- Basic calculations
- Scientific functions
- Currency conversion
- Unit conversion
- Theme customization
- AI assistant interface
- Calculation history

into one sleek web application.

---

# ✨ Features

## 🧠 Smart Calculator
- Addition
- Subtraction
- Multiplication
- Division
- Percentage
- Square root
- Decimal operations

## 📐 Scientific Functions
- sin()
- cos()
- tan()

## 💱 Currency Converter
- Convert INR to USD
- Easy input interface

## 📏 Unit Converter
- Convert centimeters to meters
- Quick conversion system

## 🎨 Theme Selector
- Dark mode
- Custom calculator themes

## 🤖 AI Assistant UI
- AI assistant input section
- Interactive smart assistant design

## 🕘 History Section
- Stores calculation history
- Clean history display area

## 📱 Responsive Design
- Mobile-friendly layout
- Modern UI/UX design
- Smooth styling and effects
